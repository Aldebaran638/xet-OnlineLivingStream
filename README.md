# xet-OnlineLivingStream

小鹅通项目：一个简易的在线直播间，支持直播与观看回放的功能

```
项目结构：
xet-OnlineLivingStream
|-documents 项目开发相关文档
|-xet-backend 后端相关文件
|-xet-frontend 前端相关文件
|-xet-operation 运维相关文件
```
